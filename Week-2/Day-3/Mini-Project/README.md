# Mini Challenge: Weather Fetcher

This task focuses on working with the Fetch API to get real-time weather data and display it on the webpage.

---

## Challenge Goal

Create a function that:

1. Fetches weather data from the API:  
   `https://api.open-meteo.com/v1/forecast?latitude=28.61&longitude=77.23&current_weather=true`

2. Displays:
   - Temperature  
   - Wind Speed  

3. Handles errors gracefully (invalid response, network issues, etc.)

---

## Key Concepts Used
- Fetch API  
- Promises and async operations  
- JSON data handling  
- Basic error handling  

---

## Expected Output
- Shows the current **temperature** and **wind speed** on the webpage.
- Displays a friendly error message if the fetch request fails.
